# Cube

## The Project

This app creates a “flying cube” with the CSS property “translate3d”. It allows the user to input three different pictures that will be repeated to create a 3-dimensional cube that floats on the screen.

## Technologies

- CSS
- Sinatra
- Ruby
- Postgres
